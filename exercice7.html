Aide-mémoire Python

print(objet) : Affiche l’objet spécifié à l’écran

abs(nombre) : Renvoie la valeur absolue du nombre

abs(-5) # Renvoie : 5

round(nombre, ndigits) : Arrondit le nombre à ndigits écimales. Si ndigits est omis, arrondit à l’entier le plus proche

round(3.14159, 2) # Renvoie : 3.14

int(objet) : Convertit l'objet en entier

int("123") # Renvoie : 123

float(objet) : Convertit l'objet en nombre à virgule flottante

float("3.14") # Renvoie : 3.14

str(objet) : Convertit l'objet en chaîne de caractères

str(123) # Renvoie : '123'

len(séquence) : Renvoie la longueur (nombre d'éléments) de la séquence

len([1, 2, 3]) # Renvoie : 3
