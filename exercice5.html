Pascal : y = 150 + 15x Érika : y = 75 + 18x
a) Imprime les deux équations.
b) Calcule le point d'égalité (résolution).
c) Trace les deux droites (GeoGebra ou matplotlib).
Indice : par où commencer ?

Identifie les quantités données et le lien entre elles (modèle de fonction ou système).

Indice : erreurs possibles

Parenthèses, noms de variables, syntaxe Python (print, input, etc.).

Solution 
from sympy import symbols, Eq, solve

x = symbols('x')
eq = Eq(150 + 15*x, 75 + 18*x)
sol = solve(eq)[0]
print(f"x = {sol}")
y = 150 + 15*sol
print(f"y = {y}")
