Énoncé :

Pascal reçoit un salaire de base de 150$ augmenté de 15$/produit vendu. Érika reçoit un salaire de base de 75$ augmenté de 18$/produit vendu.

Fais imprimer à Python les deux équations présentes dans l’énoncé.

Fais imprimer à Python le nombre de produits nécessaire afin que Pascal et Érika aient le même salaire.

Fais tracer les droites du système d’équations et identifie leur point d’intersection sur Géogebra.

Indice : par où commencer ?

Lis bien l’énoncé. Essaie de construire une fonction affine ou un système en identifiant les constantes et les variables.

Indice : erreurs possibles

Vérifie les parenthèses, les chaînes de caractères, et le nom des variables dans Python.
