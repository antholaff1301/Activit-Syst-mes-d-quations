Pour louer un canot, on paie 25 $ + 4,75 $/h.
a) Fais imprimer à Python le nombre d’heures associées si le coût est 58,25 $.
b) Fais imprimer le coût pour 13,5 h.
Indice : par où commencer ?

Identifie les quantités données et le lien entre elles (modèle de fonction ou système).

Indice : erreurs possibles

Parenthèses, noms de variables, syntaxe Python (print, input, etc.).

Solution 
# a)
y = 58.25
x = (y - 25) / 4.75
print(f"Nombre d'heures : {x}")

# b)
x2 = 13.5
y2 = 25 + 4.75 * x2
print(f"Coût pour 13.5 h : {y2}")
