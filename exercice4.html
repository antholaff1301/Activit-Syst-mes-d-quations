Exercice 4

Accueil Retour à la section
a) Fais imprimer les deux équations dont les paramètres a₁, b₁, a₂, b₂ sont entrés au clavier.
b) Déduis le nombre de solutions possibles (1, 0 ou ∞) et le type de droites (sécantes, parallèles distinctes, confondues).
Indice : par où commencer ?

Identifie les quantités données et le lien entre elles (modèle de fonction ou système).

Indice : erreurs possibles

Parenthèses, noms de variables, syntaxe Python (print, input, etc.).

Solution 
a1 = float(input("a1 = "))
b1 = float(input("b1 = "))
a2 = float(input("a2 = "))
b2 = float(input("b2 = "))

print(f"y = {a1}x + {b1}")
print(f"y = {a2}x + {b2}")

if a1 != a2:
    print("Une solution (droites sécantes)")
elif a1 == a2 and b1 == b2:
    print("Infinité de solutions (droites confondues)")
else:
    print("Aucune solution (droites parallèles)")
